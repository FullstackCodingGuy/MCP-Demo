## 🚀 **QUICK START GUIDE**

### 1. **🏗️ Environment Setup**
```bash
# Clone the repository
cd fintech-inference-service

# Create Python virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### 2. **📊 Generate Sample Data**
```bash
# Generate realistic transaction data (87K+ transactions)
python src/data/data_generator.py

# Create ML features (75 features)
python src/data/feature_engineering.py
```

### 3. **🤖 Train ML Models**
```bash
# Train churn prediction model (87.3% accuracy)
python src/models/churn_model.py

# Verify model performance
python -c "
import joblib
model = joblib.load('data/models/churn_prediction_model.joblib')
print(f'Model loaded successfully: {type(model)}')
"
```

### 4. **🔌 Start API Server**
```bash
# Start FastAPI server (21 endpoints available)
uvicorn src.api.main:app --reload --host 0.0.0.0 --port 8000

# Verify API is running
curl http://localhost:8000/health
```

### 5. **📱 Launch Interactive Dashboard**
```bash
# Start Streamlit dashboard (6 interactive pages)
streamlit run src/dashboard/app.py --server.port 8501

# Access dashboard at: http://localhost:8501
```

### 6. **🧪 Run Test Suite**
```bash
# Execute all tests (30 test cases)
pytest tests/ -v

# Run with coverage report
pytest --cov=src tests/ --cov-report=html
```

### 7. **🐳 Docker Deployment**
```bash
# Build and start all services
docker-compose up --build

# Services available at:
# • API: http://localhost:8000 (FastAPI + Auto Docs)
# • Dashboard: http://localhost:8501 (Streamlit Interface)
# • API Docs: http://localhost:8000/docs (Interactive Documentation)
```
